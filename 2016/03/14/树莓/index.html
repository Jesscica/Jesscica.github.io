    先来说说布局的几种情况

    • 左边固定，右边自适应

    两列布局

    在看方法之前，要注意一个很重要的点：凡涉及float：right,要将又浮动的元素放在前面

    1.一边固定宽度，并浮动，另一边设margin

    .left{
    float: left;
    width:200px;
}
.right{
    margin-left: 200px;
}

    right的默认宽度是100%，设置了margin后，他的宽度就变成了100%-200，此时right发现自己的宽度可以与left挤在同一行了，于是他就上来了。
    而宽度100%是相对于他的父标签来的，如果我们改变了他父标签的宽度，那right的宽度也就会变——比如我们把浏览器窗口缩小，那big的宽度就会变小，而left的宽度也就变小。

    2.一边浮动，一边设置overflow:hidden

    .left{
    float: left;
}
.right{
    overflow: hidden;
}
